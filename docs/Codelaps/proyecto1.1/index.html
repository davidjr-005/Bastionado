Esta contraseña protege TODO el acceso a la BIOS y debe establecerse PRIMERO antes que cualquier otra contraseña.

Habilitamos HDD Password, que protege el disco duro incluso si se extrae del equipo. (Lamentablemente como estamos usando un simulador de BIOS/UEFI, no se puede enseñar ya que esta opción no es compatible con la simulación)

Habilitamos la configuración de "Security Chip" el cual es requerido para el cifrado BitLocker y arranque seguro, también habilitaremos la opción "Secure Boot" para evitar arranque de sistemas no firmados o manipulados, el resto lo dejamos tal cual.

Configuramos "Boot Mode" como UEFI dado a este tiene mayor seguridad que Legacy (evita malware tipo bootkit), Deshabilitamos la configuración de "USB Boot" para evitar arranque desde dispositivos externos, Deshabilitamos también la opción "PXE Boot to LAN" para reducir el riesgo de arranque remoto no autorizado, por último nos quedaría controlar el orden de arranque para evitar el bypass que sería el Boot Order (En este caso primero Windows Boot Manager)

Desactivamos la opción "MAC Address Pass Through" para evitar spoofing o rastreo de dirección MAC, a su vez también deshabilitamos "Wake On LAN from Dock" para evitar un encendido remoto no autorizado, dejamos activada la opción "Wireless LAN" si la empresa donde está dispone de WiFi, si pudiera ir por cable o no disponen de una red WiFi, esta opción se la cambiamos, por último de esta captura depende de si trabajamos con virtualización, si no se usa se deshabilita.

Empezamos con la 2ª parte del menú de configuración:

  • Deshabilitamos "BIOS Back Flash" para impedir revertir la BIOS a versiones vulnerables.
  • Deshabilitamos "Fool Proof Fn Ctrl" para evitar confusión en combinaciones
  • Deshabilitamos "Always On USB" esto evita la actividad/carga de puertos cuando el sistema está apagado
  • El "Hotkey Mode" no afecta a la seguridad.

Por último deshabilitamos "One Key Battery" para evitar activaciones no intencionadas, para guardar los cambios y salir F10.
